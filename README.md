# Restaurant Reservation Platform

## Description
Full-stack restaurant reservation platform built with React, NestJS, TypeScript and PostgreSQL. The application supports reservation workflows, user authentication, payments and real-time interactions.

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- Axios
- React Router
- Chart.js

### Backend
- NestJS
- TypeScript
- TypeORM
- PostgreSQL
- JWT Authentication
- Passport
- Stripe
- WebSockets
- Swagger

## Features
- User authentication and authorization
- Restaurant reservation management
- Payment integration with Stripe
- Real-time interactions with WebSockets
- Data visualization with charts
- Frontend and backend separation
- REST API architecture

## Project Structure
- `frontend/` – client-side application
- `backend/` – server-side application

## Purpose
This project demonstrates my ability to build a modern full-stack application with a structured backend, relational database integration, authentication, payments and frontend-backend communication.
